# ShujaaCare

ShujaaCare is an app designed to help women in Kenya detect ovarian cancer earlier. It features an AI-powered symptom checker, easy access to screening centers, and trusted awareness materials. The platform supports patient and doctor dashboards, sign-up/login functionality, and more to empower women and healthcare providers in the fight against ovarian cancer.

This project is built using NextJS and Firebase.

To get started, take a look at `src/app/page.tsx`.
