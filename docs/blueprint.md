# **App Name**: ShujaaCare

## Core Features:

- AI Symptom Checker: AI-powered tool that analyzes user-reported symptoms and provides early risk assessment for potential ovarian cancer, offering personalized advice on next steps. The tool uses reasoning to decide which data to use.
- Awareness Resources: Provide educational resources about ovarian cancer, including risk factors, symptoms, and prevention strategies.
- Screening Center Locator: Enable users to find and locate nearby ovarian cancer screening centers across Kenya.
- Patient Dashboard: Personalized dashboard for patients to track symptoms, schedule appointments, and access medical records.
- Doctor Dashboard: Dedicated dashboard for doctors to review patient data, track treatment progress, and communicate with patients.
- User Authentication: Secure sign-up and login system for both patients and doctors to protect sensitive health information.
- Admin Panel: Admin panel to manage user accounts, update educational resources, and maintain the screening center database.

## Style Guidelines:

- Primary color: Light Blue (#ADD8E6), reflecting calmness, trust, and health. 
- Background color: Very pale blue (#F0F8FF), providing a clean and unobtrusive backdrop.
- Accent color: Periwinkle (#CCCCFF), used for interactive elements, maintaining a harmonious feel.
- Body and headline font: 'Inter', sans-serif, for clear and accessible information display.
- Use clean, minimalist icons in shades of blue and grey to represent different features and categories.
- Implement a clean and intuitive layout with a focus on ease of navigation for patients and doctors.
- Subtle transitions and animations to enhance user experience and provide feedback during interactions.
